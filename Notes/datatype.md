# Data type in javascript...
```
let x = "RIya Shukla";//String
let y = 44;//number
let z = 5.88;//number
let q = undefined;//undefined
const p = false;//boolean
let r = null;//object
console.log(x, y, z, q, p, r);
console.log(typeof (x), typeof (y), typeof (z), typeof (q), typeof (p), typeof (r));
```
